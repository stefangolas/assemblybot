# Failure Modes To Prevent

- Raw placement JSON hosted as if canonical.
- Generated visual CAD substituted for catalog parts without recorded search evidence.
- Missing `_verify` payload.
- Fastened joints represented only by coincident holes, with no fastener instances.
- Renders that are blank, all black, or missing assets.
- Static-only inspection of moving assemblies with possible swept interference.
- Project index showing a failed/prototype build as complete.
